⚡ 스킬, 어떻게 쓰나요? 3분이면 끝 · 눌러서 펼치기

스킬 = 클로드에게 붙이는 ‘전문 능력 한 묶음’이에요. 한 번 깔아두면, 평소처럼 말만 해도 클로드가 알맞은 스킬을 알아서 꺼내 써요.

1

플러그인 마켓 (제일 쉬워요)

클로드 코드 입력창에 아래 한 줄을 붙여넣고, 뜨는 목록에서 골라 설치하면 끝.

/plugin marketplace add anthropics/skills

공식·인기 스킬 대부분이 이 방식이에요.

2

수동 폴더

간단한 스킬은 ~/.claude/skills/ 폴더에 내려받기만 하면 돼요.

git clone <주소> ~/.claude/skills/이름
3

도구형

일부 스킬은 명령어 한 줄로 설치돼요.

npx claude-mem install

🗣️ 쓰는 법은 더 쉬워요. 설치하면 따로 부르는 명령어가 없어요 — 그냥 “이 엑셀 정리해줘”처럼 말하면 클로드가 알맞은 스킬을 자동으로 써요.
아래 각 카드의 ‘설치법 보기’에 그 스킬 전용 설치 명령어가 들어 있어요(복사 → 붙여넣기).

왜 만들었냐면요

클로드 스킬 정보는 죄다 영어에, 개발자 기준이라 어렵잖아요. 저처럼 겉핥기로 만드는 사람을 위해 — 수백 개 중 진짜 쓸 만한 것만 한국어로, 한눈에 보이게 정리했어요.
